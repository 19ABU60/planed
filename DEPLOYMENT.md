# PlanEd - Deployment Anleitung für Coolify

## Umgebungsvariablen für Coolify

Fügen Sie diese Variablen in Coolify unter "Environment Variables" ein:

```
JWT_SECRET=planed-secret-key-2025-secure
EMERGENT_LLM_KEY=<Ihr Emergent Key - fragen Sie im Emergent Chat>
DOMAIN_URL=http://187.77.64.225
```

## Wichtig

Die EMERGENT_LLM_KEY bekommen Sie von Ihrem Emergent Account.
